How It Works

From sensor to dashboard in seconds

The ESP32 reads sensor data and pushes it to the cloud. SilkSense AI picks it up and puts it in front of you instantly.

1️⃣

ESP32 Reads Sensors

DHT11/DHT22 captures temperature & humidity. MQ-135 monitors air quality every few seconds.

2️⃣

Data Sent to Cloud

ESP32 transmits via Wi-Fi to Firebase, MQTT broker, or your REST API endpoint in real time.

3️⃣

Dashboard Updates Live

SilkSense AI fetches live data and renders sensor cards, charts, and alerts instantly — no refresh needed.

4️⃣

Farmer Takes Action

Toggle devices remotely. Automated relays activate when thresholds are crossed. Alerts are sent instantly.
